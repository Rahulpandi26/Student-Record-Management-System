Student Record Management System (C Mini Project)

The Student Record Management System is a console-based application developed in C programming language. This project demonstrates the concept of modular programming and Makefile automation, where the program is divided into multiple source files and compiled into a single executable named student using a Makefile.

Features:

This project supports complete student record management with the following operations:

➕ Add Student Records (student_add.c)

❌ Delete Student Records (student_del.c)

👀 Display Student Records (student_show.c)

✏️ Modify Student Records (student_mod.c)

💾 Save Records to File (student_save.c)

🚪 Exit Application (student_exit.c)

🔢 Sort Student Records (student_sort.c)

🗑️ Delete Specific Records (student_delete.c)

🔄 Reverse Records Order (student_reverse.c)
